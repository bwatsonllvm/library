Talk Types Explained

Talks are labeled with normalized type badges. These same colors and styles are used throughout the library:

Keynote

High-level talks that frame direction, strategy, or major technical themes.

Technical Talk

Standard deep-dive sessions on implementation details, tooling, or engineering results.

Tutorial

Step-by-step educational sessions intended to teach workflows, tools, or APIs.

Panel

Multi-speaker discussion sessions focused on tradeoffs, perspectives, and community consensus.

Quick Talk

Shorter presentations that communicate one idea or result in a compact format.

Lightning Talk

Very short talks optimized for rapid updates, prototypes, or early ideas.

Student Talk

Talks highlighting student research or projects related to LLVM technologies.

BoF

Birds-of-a-Feather sessions: interactive discussion forums around a focused topic.

Poster

Poster-format research presentations, often exploratory or early-stage work.

Workshop

Hands-on or collaborative sessions with practical exercises and applied workflows.

Other

Sessions that do not map cleanly to the categories above.

Content Tags and Colors

Beyond talk type badges, the library also uses content tags to distinguish cross-source material:

Paper

Marks research publications and paper metadata indexed in the Papers section and global search.

Blog

Reserved for upcoming LLVM ecosystem blog posts and technical writeups that will be added soon.

Talk entries use one of the talk-type badges above; papers and blogs use content-type badges.

How Universal Search Works

  • Each main page has one search bar, and suggestions are grouped by Key Topics, Speakers + Authors, Talk Titles, and Paper Titles.
  • Selecting a suggestion can keep you on the current page (local filtering) or route you to All Work for cross-library results when that is the better match.
  • If you press Enter on free text, the query is committed immediately. Depending on intent, it either filters locally or opens global search.
  • Keyboard support: / focuses search, Arrow Up/Down navigates suggestions, Esc closes the suggestion list.

How to Use All Features

  1. Start broad: use All Work or any page search box to locate a topic, title, or contributor.
  2. Narrow with filters: Talks supports key topics, type, year, event, and resources; Papers supports key topics, year, and author targeting.
  3. Use sort + layout controls: switch between relevance, chronology, citations, and compact/card views where available.
  4. Pivot through People: open a person profile card to jump into their talks, papers, or global results.
  5. Browse by event history: use By Meeting to inspect each LLVM Developers' Meeting edition.
  6. Open detail pages: talk and paper pages include abstracts, links, metadata, and source references.
  7. Share reproducible views: active filters and searches are encoded in the URL so links preserve context.

Data and Scope Notes

  • This site curates metadata from public LLVM meeting archives and LLVM-related papers.
  • People records are merged across speaker/author name variants to reduce duplication.
  • Paper-author affiliations may be shown as published in source material; people cards intentionally omit affiliations.
  • For canonical meeting announcements and schedules, see the official LLVM Developers' Meeting archive.
